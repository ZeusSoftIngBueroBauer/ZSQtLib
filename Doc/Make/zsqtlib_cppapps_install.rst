Installing the Applications
---------------------------

After you have build the ZSQtLib binaries and the Setup(s) you may install the application(s).
This time the only application distributed by the ZSQtLib is the method trace client.

To install the Method Trace Client just double click on the set executable "ZSAppTrcMthClientSetup.exe"
and follow the instructions of the setup wizard.
